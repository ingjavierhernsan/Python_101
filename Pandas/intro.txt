What is Pandas?
Pandas is a Python library usedfor working with data sets.

It has functions for analyzing, cleaning, exploring, and manipulating data.

The name "Pandas" has a reference to both "Panel Data", and "Python Data Analysis" and was created by Wes McKinney in 2008.

Why use Pandas?
Pandas allows us to analy big data and make conclusions base on statistical theories.

Pandas can clean messy data sets, and make them readable and relevant.

Relevant data is very importan in data science.

What can Pnadas do?
Pandas gives you answers about the data. Like:
- Is there a correlation between two or more columns?
- What is average value?
- Max value?
- Min value?

Pandas are also able to delete rows that not relevant, or contains wrong values, like empty or NULL values.
This is called cleaning the data.
